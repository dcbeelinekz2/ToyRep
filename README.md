# ToyRep
Собираем дешевый 3D принтер. Стоимость минимальна, способен печатать PLA пластиком небольшие детали.

Сайт проекта: http://reprap.org/wiki/ToyREP

#### Контроллер
Arduino Mega 2560 R3 + RAMPS 1.4
